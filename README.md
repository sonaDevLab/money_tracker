# 🏦 Simple Bank App

Aplicación sencilla de **banca simulada** desarrollada con **Dart y Flutter** para practicar el desarrollo de aplicaciones móviles.

La app permite realizar operaciones básicas como **depositar dinero, retirar dinero y visualizar el saldo actual**, además de mantener un **historial de movimientos**.

---

# 📌 Descripción

Simple Bank App es una aplicación móvil básica que simula algunas funcionalidades de una cuenta bancaria.

El objetivo del proyecto fue practicar el desarrollo con **Flutter**, la gestión del **estado de la aplicación** y la creación de interfaces móviles simples.

---

# ✨ Funcionalidades

- 💰 Depositar dinero en la cuenta  
- 💸 Retirar dinero de la cuenta  
- 📊 Visualizar el **saldo total disponible**  
- 📜 Historial de movimientos realizados  
- 📱 Interfaz móvil sencilla desarrollada con Flutter

# 📱 Cómo funciona

1. El usuario introduce la cantidad de dinero que desea **depositar o retirar**.
2. El saldo de la cuenta se actualiza automáticamente.
3. Cada operación se guarda en el **historial de movimientos**.

- Uso de **Dart** para manejar la lógica de la aplicación.
- Gestión del **estado de una aplicación simple**.
- Implementación de un **historial de acciones del usuario**.
